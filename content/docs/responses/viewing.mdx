View and analyze responses as they come in.

## Responses Dashboard

Access responses:

1. Open your form
2. Click the **Responses** tab
3. View summary and individual responses

## Summary View

Get an at-a-glance overview:

### Metrics

| Metric | Description |
|--------|-------------|
| **Total Responses** | Number of submissions |
| **Completion Rate** | Started vs completed |
| **Average Time** | Time to complete |
| **Views** | Form page views |

### Charts

Each question shows aggregated data:

- **Multiple Choice** — Pie/bar chart of selections
- **Rating** — Distribution histogram
- **Scale** — Average score with distribution
- **Text** — Word cloud (Pro)

## Individual Responses

Click any response to see full details:

- All answers in order
- Submission timestamp
- Completion time
- Hidden field values
- IP address and location (if enabled)

### Response Status

Mark responses for follow-up:

| Status | Use Case |
|--------|----------|
| **New** | Unreviewed response |
| **Reviewed** | You've seen it |
| **Starred** | Important/flagged |
| **Archived** | Hidden from main view |

## Filtering & Search

### Filter Options

Filter responses by:

- Date range
- Status (new, reviewed, starred)
- Specific answer values
- Hidden field values
- Completion status

### Search

Search across all text responses:

1. Click the **Search** icon
2. Enter your search term
3. Results highlight matching responses

## Real-time Updates

Responses appear in real-time. No refresh needed.

<Callout type="info">
  Enable browser notifications to get alerted when new responses arrive.
</Callout>

## Response Actions

### Single Response

Click the **⋮** menu on a response:

- **View Details** — Full response view
- **Star** — Flag for follow-up
- **Archive** — Remove from main view
- **Delete** — Permanently remove

### Bulk Actions

Select multiple responses:

1. Check responses in the list
2. Use the bulk action bar:
   - Mark as reviewed
   - Star selected
   - Archive selected
   - Delete selected

## Partial Responses

View incomplete submissions:

1. Click **Partial** tab
2. See where people dropped off
3. Analyze abandonment patterns

<Callout type="tip">
  High drop-off on a specific question? Consider simplifying it or making it optional.
</Callout>

## Email Notifications

Get notified of new responses:

1. Go to Settings → Notifications
2. Enable **Email Notifications**
3. Choose frequency:
   - Instant (each response)
   - Daily digest
   - Weekly summary

### Notification Content

Emails include:
- Response summary
- Key field values
- Direct link to response
- Quick stats

## Collaboration

### Team Access <span className="text-xs bg-primary/10 text-primary px-2 py-1 rounded">Pro</span>

Share access with your team:

1. Go to Settings → Team
2. Invite by email
3. Set permissions:
   - **View** — Read-only access
   - **Edit** — Modify form and responses
   - **Admin** — Full control

### Comments

Add internal notes to responses:

1. Open a response
2. Click **Add Comment**
3. Tag teammates with @mention

## Data Retention

Responses are stored securely:

| Plan | Retention |
|------|-----------|
| Free | 30 days |
| Pro | 1 year |
| Business | Unlimited |

<Callout type="warn">
  Export important data before retention limits. Deleted responses cannot be recovered.
</Callout>

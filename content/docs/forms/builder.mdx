The Tyform form builder is designed to be intuitive yet powerful.

## Interface Overview

The form builder is divided into three main sections:

### Left Panel — Question List

View and reorder all your questions. Drag and drop to rearrange.

### Center Panel — Live Preview

See exactly how your form will look to respondents in real-time.

### Right Panel — Question Editor

Configure the selected question's settings and options.

## Building Your Form

### Adding Questions

Click the **"+"** button or press `Q` to add a new question. Choose from:

| Type | Description |
|------|-------------|
| Welcome Screen | Introduce your form |
| Short Text | Single line input |
| Long Text | Multi-line textarea |
| Email | Email with validation |
| Number | Numeric input |
| Phone | Phone number input |
| URL | Website URL input |
| Multiple Choice | Single selection |
| Checkboxes | Multiple selection |
| Dropdown | Compact single selection |
| Rating | Star or number rating |
| Scale | Linear scale (1-10) |
| Date | Date picker |
| File Upload | Accept files (Pro) |
| Statement | Display text without input |
| Thank You Screen | Confirmation message |

### Editing Questions

Click any question to select it. The right panel shows:

- **Question Text** — Main question copy
- **Description** — Optional helper text
- **Settings** — Question-specific options
- **Validation** — Required, min/max, etc.

### Reordering Questions

Drag questions in the left panel to reorder them.

<Callout type="tip">
  Use keyboard shortcuts for faster editing: `Q` to add, `Delete` to remove, arrow keys to navigate.
</Callout>

## Question Settings

### Common Settings

All question types share these settings:

| Setting | Description |
|---------|-------------|
| Required | Must answer to proceed |
| Description | Helper text below question |
| Placeholder | Example text in input |
| Variable | Reference in logic rules |

### Type-Specific Settings

Different question types have unique settings:

**Multiple Choice / Checkboxes:**
- Allow "Other" option
- Randomize order
- Multiple columns

**Rating:**
- Scale (1-5 or 1-10)
- Icon style (stars, hearts, etc.)

**File Upload:**
- Allowed file types
- Maximum file size
- Multiple files

## Preview Mode

Click **Preview** to test your form:

- Navigate through all questions
- Test validation rules
- Check logic branching
- View on desktop and mobile

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Q` | Add question |
| `Delete` | Delete question |
| `↑` / `↓` | Navigate questions |
| `Cmd + S` | Save form |
| `Cmd + P` | Preview form |
| `Cmd + Z` | Undo |
| `Cmd + Shift + Z` | Redo |

## Auto-Save

Your form saves automatically as you edit. Look for the save indicator in the header.

<Callout type="info">
  Changes are saved locally until published. Publish to make changes live.
</Callout>
